# Creative Services Platform (CaaS)

Una plataforma moderna de servicios creativos con sistema de suscripción basado en créditos.

- Estrategia de marca

- Roadmap de producto

### Branding

- Aplicaciones de marca

### Producto Digital
- Diseño UI/UX
- Interfaces web y móvil
- Diseño de producto
- Prototipos interactivos

## Sistema de Créditos

### Estructura de Precios
- 40 créditos: €2,620/mes
- 80 créditos: €5,240/mes
- 120 créditos: €7,860/mes
- 160 créditos: €10,480/mes
- 200 créditos: €13,100/mes

### Complejidad y Uso
1. Baja Complejidad (1 crédito/hora)
   - UI Design
   - Presentaciones
   - Piezas gráficas

2. Media Complejidad (1.25 créditos/hora)
   - Dirección de arte
   - Design systems
   - Motion design

3. Alta Complejidad (1.5 créditos/hora)
   - Estrategia de marca
   - Diseño de producto
   - Creative direction
